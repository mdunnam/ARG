# Sleep Clinic ARG Reference Links

These projects are useful references for the Somnatek sleep clinic ARG. None are exact matches, but each contains techniques worth studying: analog horror presentation, believable fictional institutions, web-native clue hunting, live audience participation, or document-based worldbuilding.

## Reference Projects

| Project | Link | Why It Matters |
|---|---|---|
| Marble Hornets | https://www.youtube.com/user/MarbleHornets | Early web-native horror ARG with community solving, hidden clues, and an escalating sense that viewers were close to the danger. Heavily video-based, but useful for pacing and audience paranoia. |
| Local 58 | https://www.youtube.com/@LOCAL58TV | Strong analog horror reference using institutional broadcast language, public warnings, and restrained presentation. Useful for tone, even though it is video-centered. |
| Gemini Home Entertainment | https://www.youtube.com/@GEMINIHOMEENTERTAINMENT | Analog horror built from fake educational and public-service media. Useful for how mundane formats can carry dread. |
| The Sun Vanished | https://twitter.com/TheSunVanished | A live social-media ARG that felt ongoing and reactive. Useful for studying cadence, audience speculation, and real-time updates. |
| Alan Resnick / alantutorial | https://www.alantutorial.com/ | Surreal web horror with mundane presentation turning unstable. Useful for escalation and unsettling ordinary formats. |
| This House Has People In It | https://www.adultswim.com/videos/infomercials/this-house-has-people-in-it | Mundane domestic footage and associated web materials become deeply wrong through implication and layered context. Useful for hidden-site and supplemental-material thinking. |
| Welcome Home | https://www.clownillustration.com/welcome-home | Polished website-as-artifact horror project with layered pages, source clues, and an innocent public surface hiding darker material. Useful for web exploration design. |
| I Love Bees | https://ilovebees.com/ | Classic public ARG with distributed participation, phone/audio elements, and event-based progression. Useful as a large-scale ARG structure reference. |

## Most Relevant Comparisons

### Most Useful for Tone

- Local 58
- Gemini Home Entertainment
- Mystery Flesh Pit National Park

These show how institutional formats can feel disturbing without needing overt monsters or direct explanation.

### Most Useful for Web Structure

- Welcome Home
- This House Has People In It
- I Love Bees

These are good references for hidden pages, supplemental materials, cross-site clues, and layered audience discovery.

### Most Useful for Live Cadence

- The Sun Vanished
- I Love Bees
- Marble Hornets

These are useful for studying how a project can feel active over time and how the audience changes the energy of the work.

### Most Useful for Community Puzzle Culture

- Cicada 3301
- Petscop
- Welcome Home

These are useful for seeing how players organize, document clues, theorize, and keep mysteries alive between drops.

## How Somnatek Differs

The Somnatek concept should not copy any one reference. Its lane is distinct because it combines:

- A fake sleep clinic / patient portal.
- Medical and research bureaucracy as horror.
- A no-video, document-first presentation.
- Multiple believable entities: clinic, forum, university archive, county records, corporate successor.
- Live daily / weekly updates.
- Opt-in visitor IDs and appointment language.
- A gradual shift where the player is treated as part of the experiment.

The goal is not to feel like analog horror fan fiction. The goal is to feel like an abandoned medical archive that starts quietly classifying the people who investigate it.
